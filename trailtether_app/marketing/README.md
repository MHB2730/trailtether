# Trailtether Marketing Materials

This folder contains launch-ready marketing copy, store listing text, social copy, a static landing page, and rendered promotional graphics for Trailtether.

## Files

- `marketing_pack.md` - positioning, taglines, elevator pitch, feature copy, email copy, ad copy, press boilerplate, and screenshot captions.
- `store_listings.md` - Google Play and Microsoft Store listing drafts.
- `social_launch_kit.md` - LinkedIn, Instagram, Facebook, X/Twitter, short video script, and hashtags.
- `landing_page.html` - static one-page marketing site using the app icon and mountain hero image.

## Graphics

- `social/trailtether_launch_square.png` - 1080 x 1080 square post.
- `social/trailtether_story.png` - 1080 x 1920 vertical story.
- `social/trailtether_banner.png` - 1600 x 900 widescreen banner.
- `social/trailtether_flyer_a4.png` - A4 flyer artwork.

## Core Message

Trailtether helps hikers plan smarter, hike safer, and stay connected with route planning, GPX imports, offline-aware maps, live tracking, teams, weather, field intel, caves, shelters, and SOS tools.
