# Trailtether

Flutter hiking app with local trail data and Supabase-backed auth, profiles, reviews, incidents, team chat, and GPX uploads.

## Quick Start

```powershell
flutter pub get
flutter analyze
flutter build apk --release
flutter build windows --release
```
