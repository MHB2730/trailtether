# 🎮 Mission Control Guide

The "Mission Control" dashboard is the primary interface for operators.

## 🕹️ Controls
- **Mode Toggle**: Switch between **Hybrid (Satellite)**, **Street**, and **Terrain** views.
- **3D Weather**: Toggle precipitation radar and cloud layers.
- **Reset View**: Returns the camera to the default mission center.
- **Fit All**: Zooms out to show every tracked asset/route.

## 📋 Information Panels
- **Sidebar**: Searchable list of all active routes and their high-level stats (Dist/Max Ele).
- **Detail Sheet**: Opens on selection. Contains:
  - Full route name and ID.
  - Interactive Elevation Profile.
  - Total Elevation Gain.
  - Difficulty/Status Badge.

## ⌨️ Shortcuts & Interactions
- **Left Click**: Select unit/route.
- **Right Click + Drag**: Tilt and rotate 3D view.
- **Mouse Wheel**: Zoom.
- **Hover**: View specific coordinate data under the cursor.
