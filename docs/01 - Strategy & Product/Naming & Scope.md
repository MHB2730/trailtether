# Naming & Scope

Use this note to avoid brand confusion.

## Current Working Definition

- **Hilltrek**: parent brand, public site, commerce, admin, company identity.
- **Trailtether**: hiking app/platform inside Hilltrek, including mobile app and Windows command centre.

## Recommended Usage

- Use **Hilltrek** when talking about the business, website, admin, launch, or brand.
- Use **Trailtether** when talking about the app, codebase, Supabase data, telemetry, maps, and safety workflows.
- Use **Hilltrek / Trailtether** when the context spans both.
