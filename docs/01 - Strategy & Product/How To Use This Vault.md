## Note Hygiene

- Keep one canonical note per concept.
- Use wiki-links for related concepts.
- Add source file paths to notes that summarize code.
- Move stale notes to `99 - Archive`; do not leave duplicates beside current notes.
- Use status values: `current`, `draft`, `wip`, `legacy`, `archived`.
