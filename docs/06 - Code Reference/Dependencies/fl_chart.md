# fl_chart

`fl_chart: ^0.69.0`

Charts library. Used for the elevation profile chart.

## Primitives used

- `LineChartData`, `LineChartBarData` — line charts
- `FlSpot` — single point
- `BorderSide`, `FlGridData`, `FlTitlesData` — styling

## Consumers

- `ElevationChart` widget in `widgets/trail/elevation_chart.dart`
- `_MiniElevCard` in [[TTMapScreen]]'s recording panel
